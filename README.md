# TaskFlow

TaskFlow is a basic task management application.

## Repository Structure
- src/ : source files
- docs/ : documentation files
- tests/ : testing-related files

## Purpose
This repository demonstrates Git and GitHub workflow practices for DevOps learning.

## Workflow
Development is done using feature branches, bug-fix branches, pull requests, issues, milestones, version tagging, and releases.

TaskFlow is a simple task management system.
It includes modules such as login, dashboard, and task CRUD operations.
This project is maintained using Git and GitHub workflow practices.
